Developer Gary's Story.

All of the content here is arranged using CSS Flexbox. I spent a total of 13 hours in various Team Treehouse, and assorted other resources to get this far. The three hours with Professor Jimenenez on flexbox styling really got me interested and sent me down a (fun) learning rabbit hole that still hasn't reached bottom!

The rules and settings within Flexbox are really fun and easy to work with. In the past I built smaller webpages using floats and grids and so on; what a headache that was!

This website landing page took me about 3 hours to build, all thanks to the power of Flexbox and a few powerful @media rules.

This page is optimized for mobile screens first.

Site Highlights

Flexbox: The Flexbox Layout (Flexible Box) module provids a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic (thus the word “flex”).

CSS Media Queries: The @media rule, introduced in CSS2, makes it possible to define different style rules for different media types, such as styling for mobile and desktop at the same time.
